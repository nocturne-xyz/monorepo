# Nocturne Monorepo

## Installation

- Run `yarn install`
- Run `git submodule init`
- Run `git submodule update`
- Run `scripts/install_base_deps.sh`
- Run `scripts/install_foundry_deps.sh`

## Build

- Run `yarn build`

## Run Tests

- Unit tests + contract invariant/fork tests: `yarn test:unit`
- E2E tests: `yarn test:e2e`
